# Excel アプリ

Next.js と SheetJS で構築したブラウザベースの Excel ビューア・エディター・共有ツール。

## 概要

Excel ファイルをブラウザで直接開け、ソフトウェアをインストールする必要がありません。スプレッドシートを表示・編集し、URL で共有できます。

## 機能

- **ファイルを開く**: ドラッグ&ドロップまたはクリックで `.xlsx`, `.xls`, `.xlsm`, `.csv` をアップロード
- **編集**: セルをダブルクリックして編集。数式バーで数式を入力 (`=SUM(A1:B5)`)
- **書式設定**: 太字、斜体、下線、フォントサイズ、色、配置、折り返し
- **数式**: SUM, AVERAGE, COUNT, MAX, MIN, IF, VLOOKUP
- **複数シート**: タブナビゲーション、シートの追加・名前変更・削除・並べ替え
- **グラフ**: 選択データから棒グラフ、折れ線グラフ、円グラフを挿入
- **検索・置換**: Ctrl+F で検索、次/前へ移動、すべて置換
- **エクスポート**: `.xlsx`, `.xls`, `.csv` でダウンロード、または PDF で印刷
- **共有**: ワンクリック URL 共有 — ワークブックは URL に圧縮して埋め込み
- **元に戻す・やり直す**: Ctrl+Z / Ctrl+Y で完全な履歴
- **キーボードショートカット**: 矢印キーナビゲーション、Ctrl+C/X/V、F2、Delete など

## サポートされているファイル形式

| 形式 | 読込 | 書出 |
|------|------|------|
| `.xlsx` | ✅ | ✅ |
| `.xls` | ✅ | ✅ |
| `.xlsm` | ✅ | ✅ |
| `.csv` | ✅ | ✅ |

## Vercel にデプロイ

1. このリポジトリをクローンまたはダウンロード
2. 依存関係をインストール:
   ```bash
   npm install
   ```
3. デプロイ:
   ```bash
   npx vercel
   ```

完了です。基本的な使用には環境変数は必要ありません。

**オプション**: 大きなファイル共有の場合は、Vercel Blob Storage から `BLOB_READ_WRITE_TOKEN` を追加してください。

## ローカル開発

```bash
npm install
npm run dev
```

[http://localhost:3000](http://localhost:3000) を開きます。

## 注記

- これは完全な Excel 代替品ではありません — 複雑なマクロ (VBA) はサポートされていません
- 数式サポートは一般的な関数をカバーしています: SUM, AVERAGE, COUNT, MAX, MIN, IF, VLOOKUP
- AES 暗号化を使用したパスワード保護ファイル (最新の .xlsx) はブラウザで開くことができません
- 共有 URL にはワークブック全体のデータが含まれます。非常に大きいファイル (圧縮後 >100KB) は URL で共有できません
